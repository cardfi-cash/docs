# Order status check

## Path

```
merchain/order/check/id/oid
```

## Method

```
GET
```

## Params

```
id : The merchant set
oid : Order Id from apply or deposite interface
```

## Query

```
null
```

## Success Response

```
{
    "code": 200,
    "data": []
}
```

## Failed Response

```
{
    "code": 200,
    "data": false
}
```
