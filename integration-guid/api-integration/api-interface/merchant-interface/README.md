# Merchant Interface

## Base Path

```
/merchant
```

#### Eg :&#x20;

Ping : `https://api.cardfi.cash/merchant/ping`

### Auth Rules

Add header to all your request about `merchain` role

| Header | Value   |
| ------ | ------- |
| token  | API-KEY |

You can apply your API-KEY on CardFi Bot : [http://t.me/cardfi\_bot](http://t.me/cardfi_bot)
