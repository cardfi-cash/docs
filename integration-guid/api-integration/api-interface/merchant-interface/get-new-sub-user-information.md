# Get new sub-user information

## Path

```
/merchain/user/info/:id
```

## Method

```
GET
```

## Params

```
id : The merchant set
```

## Query

```
null
```

## Success Response

```
{
    "code": 200,
    "data": []
}
```

## Failed Response

```
{
    "code": 401,
    "error": "Permission deny . Please check if API-KEY correct ."
}
```
